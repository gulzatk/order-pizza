# Order Pizza

#### Object Oriented JavaScript Independent Project for Epicodus, 02.11.2018

## Description

This is a Webpage, where you can order a own build pizza choosing topping and size.

## Setup/Installation Requirements

Clone Repository
Open index.HTML
